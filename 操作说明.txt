  接下来一次性配置 VSCode：

  1. 没装 VSCode → 去 https://code.visualstudio.com/ 装（默认 Next 即可）
  2. 用 VSCode 打开 F:\paperwritting\paper\ 整个文件夹（File → Open Folder）
  3. 第一次打开会右下角弹窗 "There is an available extension for this folder (LaTeX Workshop)" → 点 Install
    - 没弹窗就左侧 Extensions 面板 → 搜 latex workshop → 装

  以后工作流：

  ┌─────────────────────────────────┬─────────────────────────────────────────────────────────┐
  │              操作               │                         快捷键                          │
  ├─────────────────────────────────┼─────────────────────────────────────────────────────────┤
  │ 编辑 .tex 后自动重编 + 看新 PDF │ Ctrl+S                                                  │
  ├─────────────────────────────────┼─────────────────────────────────────────────────────────┤
  │ 强制重编（重置缓存）            │ Ctrl+Alt+B 选 pdflatex -> bibtex -> pdflatex x 2        │
  ├─────────────────────────────────┼─────────────────────────────────────────────────────────┤
  │ .tex 行跳转 PDF 对应位置        │ Ctrl+Alt+J 输入行号                                     │
  ├─────────────────────────────────┼─────────────────────────────────────────────────────────┤
  │ Ctrl+点击 PDF                   │ 反向跳回 .tex                                           │
  ├─────────────────────────────────┼─────────────────────────────────────────────────────────┤
  │ 关 PDF tab                      │ Ctrl+W 单独关                                           │
  ├─────────────────────────────────┼─────────────────────────────────────────────────────────┤
  │ 临时清 build/                   │ Ctrl+Shift+P → LaTeX Workshop: Clean up auxiliary files │
  └─────────────────────────────────┴─────────────────────────────────────────────────────────┘

  关键效果：

  - 编译产物全部进 F:\paperwritting\paper\build\，主目录保持只有源文件（你已经清理过，VSCode 不会再搞乱）
  - 报错时左侧行号旁边有红 X，点击跳转到出错行
  - PDF 实时在右 tab 刷新，所见即所得

  首次使用提示： 第一次 Ctrl+S 会让 LaTeX Workshop 跑完整 4 遍 recipe（pdflatex → bibtex → pdflatex × 2），大约 5-10
  秒。之后热重编更快。

  可以正常写论文了。改完任何 .tex / .bib 直接保存，右边 PDF 自动更新。



以后要让 PDF 自动更新，存完 .tex 文件后按 Ctrl+Alt+B 手动触发编译，或者在 VSCode 左侧 TEX 面板点 "Build LaTeX project"。LaTeX Workshop 的自动保存编译有时候不灵敏，手动最稳。
